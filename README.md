<h1>Fundamental Financial Analysis</h1>

<p>
This project demonstrates a basic fundamental analysis using income statement data.
Key financial ratios are calculated and combined into a simple rule-based
fundamental score.
</p>

<h2>Metrics Included</h2>
<ul>
  <li>Revenue Growth (%)</li>
  <li>Net Margin (%)</li>
  <li>Operating Margin (%)</li>
  <li>EBITDA Margin (%)</li>
</ul>

<h2>How to Run</h2>
<pre><code>pip install pandas</code></pre>
<pre><code>python fundamental_analysis.py</code></pre>

<p>
When prompted, enter the full path to your income statement CSV file.
</p>
